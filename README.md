# Tout Droit – Landing page

Pour afficher la page en local et avoir un aperçu :

1. Ouvre un terminal dans ce dossier (`/workspace/test`).
2. Lance un petit serveur web :
   ```bash
   python3 -m http.server 8000
   ```
3. Dans ton navigateur, visite [http://localhost:8000/index.html](http://localhost:8000/index.html).

Tu verras la page d’accueil exactement comme elle sera servie en ligne.
